Variable Declarations

Do not use var anymore

Const

  • If you do not expect variable to every change use const

Let

  • If you expect variable to change use let

Template Strings or template literals

  • Use backticks
  • no more need for a plus sign and you change that to ${}
  • helps with readability

Fat Arrow Functions

  • If you have a single argument you can drop the ()
  • if you have one statement you can put it on one line and remove the parenthese
